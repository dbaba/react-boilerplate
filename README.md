server side rendering (Removed Flux)

ES6 style

* react
* react-router
* express
* browserify
* babel

## setup
```
npm install
```

## run server
```
npm start
```

## run client
```
npm run-script client
```

## manual build
* create bundle.js
```
npm run build
```

## test (with Jest)

```
npm test
```
